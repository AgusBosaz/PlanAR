# PlanAR - Página de Eventos

## Descripción

Este proyecto consiste en el desarrollo de una nueva funcionalidad para la plataforma **PlanAR**, realizada de manera colaborativa utilizando **Git** y **GitHub**.

La funcionalidad implementada es una página llamada **eventos.html**, donde los usuarios pueden visualizar distintos eventos disponibles y realizar acciones interactivas mediante JavaScript.

---

## Objetivos del Sprint

* Trabajar colaborativamente mediante Git y GitHub.
* Aplicar conocimientos de HTML, CSS y JavaScript.
* Implementar prácticas básicas de QA e Infraestructura.
* Desarrollar una página de eventos con funcionalidades dinámicas.

---

## Funcionalidades Implementadas

### Estructura y Diseño

La página incluye:

* Título principal.
* Descripción breve.
* Tres tarjetas de eventos.
* Botón "Reservar lugar".
* Botón "Mostrar categorías".
* Botón "Volver al inicio".

Cada tarjeta de evento muestra:

* Nombre del evento.
* Fecha.
* Lugar.

---

### Estilos CSS

Se utilizaron las siguientes características:

* Flexbox para la distribución de las tarjetas.
* Colores personalizados.
* Bordes redondeados (`border-radius`).
* Sombras (`box-shadow`).
* Efecto hover sobre las tarjetas.

---

### Funcionalidades JavaScript

#### Reservar lugar

Al presionar el botón **"Reservar lugar"**, se muestra un mensaje dentro de la página:

> Reserva realizada correctamente

#### Mostrar categorías

Al presionar el botón **"Mostrar categorías"**, se muestran las siguientes categorías cargadas desde un array de JavaScript:

* Conciertos
* Ferias
* Deportes
